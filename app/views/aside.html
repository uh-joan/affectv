Description

Advertisers

New

To add a new one type a name.

To submit hit Enter or click +. Esc to exit.

To search type the name, if the advertiser already exists, it will be loaded.

Details

Click on advertiser name to view properties.

The number shows the pixels that belong to the advertiser.

When hovering over the advertiser name you can click on edit or delete.

Edit

When edit is clicked the advertiser details are loaded.

To submit the new data hit Enter or +. Esc to exit.

If the advertiser name is cleared and then submitted the advertiser will be removed.

Pixels

To add a pixel click on 'add new pixel +'. Hit Enter to add or Esc to cancel.

To edit an existing pixel double click on the pixel name. Hit Enter to edit or Esc to cancel

Fires

Hover over a pixel to show the fires, if they exist.

Click on fires or 'see more' to open a navigation page with a graph for the fires.
